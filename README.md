# Voluntheroes
Voluntheroes is a web application that allows volunteers to help elderly people with day to day task they can not always fullfill. The functionality of this application is a user can sign up to be a volunteer. That volunteer can then see their task that are either available, claimed by other volunteers, and their own task they have claimed.

## Getting Started
First - Fork and clone this repository
Second - Navigate into the directory where you cloned the repository.
